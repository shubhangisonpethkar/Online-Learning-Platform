
# Progress Tracking Module

## Purpose

Track the learning progress of each enrolled student.

## Module Flow

Student
   ↓
Enrolled Courses
   ↓
Completed Lessons
   ↓
Calculate Progress Percentage
   ↓
Update Dashboard
   ↓
Course Completion
   ↓
Certificate Generation

## Future Features

- Percentage Progress
- Lesson Completion
- Resume Learning
- Certificate Status
